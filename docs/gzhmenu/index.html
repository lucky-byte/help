公众号菜单收款指南

商户可以在自己公众号底部的菜单中增加一个子菜单，例如报名缴费，用户选择菜单进入付款页面。

注意

在公众号内配置收款菜单需要满足几个条件：

  1. 公众号必须是服务号。登录公众号后，在页面的右上角可以看到是服务号还是订阅号。
  2. 有公众号管理员账号，可以登录公众号后台进行各种操作。
提示

如果您不熟悉微信公众号的后台配置或遇到困难，可以委托技术支持人员帮您配置，您需要提供公众号管理员账号。
