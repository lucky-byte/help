强调#

强调, 例如斜体, 用 星号 或者 下划线。

高度强调, 例如粗体, 用 双星号 or 双下划线。

组合强调用 星号 和 下划线。

删除用两个波浪号，删除文字

列表#

  1. 第一个有序列表项
  2. 另一项
    • 无序子项
  3. 实际的编号不用关心，只要是一个数字就行了
    1. 有序子项
  4. 再加一项
  • 无序列表可以用星号
  • 或者减号
  • 或者加号
